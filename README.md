# AlterEgo — онлайн-платформа для управления салоном кастомной одежды

Тема курсовой: *онлайн-платформа для управления салоном кастомной одежды*. В проекте **нет пошива и примерок**: студия **только декорирует** готовые вещи (роспись, вышивка, стразы, принт и т.д.), заказ оставляют и забирают/отправляют готовое. **Корпоративы** — тираж с брендированием. Учёт заказов, **материалов для декора** (не ткани на крой), статусов и **связи с клиентом** (созвоны по макету, пересылка вещи туда–обратно, визиты в студию).

Стек: React (MUI) + Redux Toolkit, Node.js (Express), PostgreSQL (Prisma), REST API.

## Роли

- **CLIENT** — оформление и отслеживание **своих** заказов, мастер-классы (информационная страница), профиль.
- **ADMIN** — управление каталогом и материалами, работа с заказами всех клиентов, панель аналитики и отчёты PDF/DOCX.

## Запуск

1. PostgreSQL: создайте БД, например **`salon_kastomnoy_odezdy`** (имя может быть любым — оно должно совпадать с `DATABASE_URL` в `.env`).
2. `cd server` → `copy .env.example .env` — укажите `DATABASE_URL` и `JWT_SECRET`.
3. Установка и БД:

```bash
cd server
npm install
npx prisma generate
npx prisma db push
npm run db:seed
npm run dev
```

4. В другом терминале:

```bash
cd client
npm install
npm run dev
```

Клиент: http://localhost:5173 (прокси `/api` → http://localhost:4000).
